\# Tarea Diseño - LTI ATS



Este proyecto corresponde al diseño inicial de un sistema ATS (Applicant Tracking System) moderno.



\## 🎯 Objetivo

Diseñar un sistema que:

\- Mejore la eficiencia de HR

\- Permita colaboración en tiempo real

\- Incorpore automatizaciones

\- Utilice IA en tareas de recruiting
